# mariadb

[![Build](https://github.com/gsmlg-ci/mariadb/actions/workflows/build.yml/badge.svg)](https://github.com/gsmlg-ci/mariadb/actions/workflows/build.yml)

Docker image published to:
- `docker.io/gsmlg/mariadb`
- `ghcr.io/gsmlg-dev/mariadb`

## Usage

```bash
# From Docker Hub
docker pull gsmlg/mariadb:latest

# From GitHub Container Registry
docker pull ghcr.io/gsmlg-dev/mariadb:latest
```

## Build

```bash
docker build -t gsmlg/mariadb .
```
